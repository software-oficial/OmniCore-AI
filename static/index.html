Bienvenido al Meta-Orquestador

OmniCore-AI es la capa de inteligencia que gestiona tu infraestructura de negocio. Desde aquí puedes registrarte como desarrollador, obtener tus credenciales y empezar a desplegar módulos de Stock, Ventas y WhatsApp.

🚀 Guía rápida de inicio

  1. Ve a Registro de Agente y crea tu identidad.
  2. Copia tu TOKEN generado.
  3. Usa el token en el header Authorization: Bearer YOUR_TOKEN para llamar a la API.

Registro de Agente

Crea una identidad de agente para comenzar a operar en modo LEARNING (Sandbox).

Documentación de la API

OmniCore-AI utiliza una arquitectura basada en comandos. Cada acción es un comando ejecutado vía el Gateway.

POST /api/gateway/execute

El endpoint principal para ejecutar cualquier comando de negocio.

Header: Authorization: Bearer {token} Body: { "command": "stock.add", "params": { ... } }
GET /api/agent/manifest

Retorna la ontología actual del sistema y la lista de comandos disponibles.
